Cleaning Surface Robot

My capstone team was awarded the opportunity to develop a robotic arm that can clean a set of surfaces in a medical environment. The significance of having a robot that can clean surfaces is to reduce manual labour and have consistency in disinfecting without creating errors. The main objective of this project was to create a mobile robot with a manipulator arm that can autonomously and manually navigate in an environment to clean a set of surfaces. To accomplish this, the project had a four phase mechatronic design process. First was to develop a concept sketch of a robot that would meet all the analysed requirements. The next step was to develop a simple robot using design iterations to be tested in a simulated environment using Webots. Once the robot was tested, a CAD was created using SolidWorks, with details on the robot arm and the mobile base of the robot. This final CAD model was given a name of EC-P1. Lastly, a small model prototype was constructed through a series of iterations. As a result the EC-PI robotic disinfectant manipulator was designed.

Team Members: Ani B., Saumya S. and Sebastian S.



Robot Car

The purpose of this project was to gain in-depth hands-on experience with microcontrollers by creating a robot car that can navigate autonomously and with manual input through a set path. The requirements for this project was to manually control the car, program the car to navigate autonomously and to successfully avoid an arbitrary obstacle positioned on its path. This project consisted of implementing mechanical, electrical and software components. The electrical components of this design consisted of using a servo motor for directional control and stepper motor for linear movement. Additionally an ultrasonic distance sensor was used to detect the obstacle in front of the car. In terms of the mechanical component common household materials such as popsicles sticks, bottle caps and tape/glue to assemble the components.

Team Members: Ani B., Talin D. and Kevin A.
